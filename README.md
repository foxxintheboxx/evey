# Evey

### ngrok
Ngrok sets up secure tunnels to our localhost i.e. Ngrok gives web accessible URLs and tunnels
all traffic from that URL to our localhost!

### Useful commands
```bash
. venv/bin/activate
./ngrok http 8000
./db_create.py
heroku run python manage.py db upgrade
heroku pg:reset
```
